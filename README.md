# Опис варіанту
1. Завдання №10: Визначити клас будівельного блоку (по типу блоків в minecraft), який складається мінімум з 5-и полів
2. Сортування: Відсортувати масив, за одним з полів за зростанням, а за іншим — за спаданням
3. Пошук: Знайти в масиві об’єкт, який ідентичний заданому.
